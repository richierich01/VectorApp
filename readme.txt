Find enclosed a Visual Studio Solution

-Open in Visual Studio 2019

-The VectorDatabase Project, creates a database and populates with sample data
		-After publishing the database, copy the connection string to appsettings.json file in the VectorApp and also to the WebConfig in the Vector WebForms Project
		
-VectorWebForms Project shows the student records in a simple grid
		
-VectorApp is the WebAPI Service which provides data the angular application hence run this service before starting the angular application

-In the VectorApp directory is Angular Application in the VectorUI directory
	-Open this folder in Visual Studio Code
	-Open the Terminal in Visual Studio Code
	-Run npm install
	-Run npm start to start the Angular Application.
